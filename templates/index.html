Manual Survival Analysis (Log-rank)

Enter two groups of survival records. Use one row per participant with: time,event (or time event). Event should be 1 for event and 0 for censored.

Manual mode: uses your records directly (no image extraction).

Upload Kaplan-Meier Image (LLM API Extraction)

Upload mode: uses the OpenAI Responses API to infer approximate structured data from the KM figure, then computes an approximate two-group log-rank result from reconstructed records.

API keys are read only from OPENAI_API_KEY environment variables. This page never asks you to paste keys.
